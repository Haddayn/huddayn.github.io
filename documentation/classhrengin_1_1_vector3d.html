Vector3d<T> hrengin::Vector3d< T >::getHorizontalAngle ( ) const
inline

Get euler angles that when applied to a (0,0,1) direction vector would make it point in the same direction as this vector.

Original author of this method is Arras from the Irrlicht forums

Returns
A rotation vector containing the X (pitch) and Y (raw) rotations in degrees, of this vector. The Z (roll) rotation is always 0, since two rotations are sufficient. (does a vector even have roll rotation? I don't think so)
